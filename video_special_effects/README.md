## Local Setup
MacOS Sequoia 15.2 - Arm64 
Visual Studio Code
Cmake 3.31.3

To compile the code. Simple run the following commands from root.
```
mkdir build
cd build
cmake ..
make
```
The executable binaries should then appear in the build folder.
